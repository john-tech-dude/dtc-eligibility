# Visual QA Checklist

Run after layout/CSS changes or before deploy. Prefer **hard refresh** (Cmd+Shift+R) and both **light** and **dark** themes.

Serve locally first:

```bash
python3 -m http.server 8000
# http://127.0.0.1:8000/
```

Automated smoke (when Node deps installed):

```bash
npm test
```

---

## A. Hub (`index.html`)

- [ ] Page loads; theme toggle works
- [ ] Document cards link to live modules (not 404)
- [ ] Search (if present) returns eligibility / CUSIP / SF 28 results
- [ ] Mobile (~390px): no horizontal page scroll; nav usable

## B. Eligibility questionnaire (`pages/forms/dtc-eligibility-questionnaire.html`)

### Chrome / structure
- [ ] Shared styles apply (fonts, colors match other guides)
- [ ] `styles/forms-eligibility.css` + `teaching.css` load (no unstyled flash of broken layout)

### OA Canon (`#oa-crosswalk`)
- [ ] Section renders (not the “could not load JSON” error) — **must use HTTP**, not `file://`
- [ ] Filters: All / Timelines / Crosswalk / Rules switch content
- [ ] Then (legacy) vs Now (OA) rows readable in dark mode

### Scenarios (`#scenarios`)
- [ ] Tab A–D switch panels; only one panel visible
- [ ] Keyboard ← → moves tabs; `aria-selected` updates
- [ ] Deep link `#sc-b` opens scenario B
- [ ] Hotspots inside panels still open modals

### CUSIP Information (`#cusip-table`)
- [ ] Six headers; pre-filled rows sit **under** correct columns
- [ ] Last column (Offering Price) is **inside** the facsimile border (not to the right)
- [ ] Values: `037833AA5` | `5.250%` | maturity | principal | `T` | `99.50` on row 1
- [ ] Mobile: stacked label/value pairs; no phantom 7th column

### Facsimile form
- [ ] Two-column issue/contact grid stacks on mobile
- [ ] Field-diff chips (Legacy / Updated) open modals
- [ ] Print preview: scenarios show all panels; banners hide

### Quiz
- [ ] Select answers; feedback appears; score updates
- [ ] Reset (if present) clears state

## C. DTC Guide (`pages/guides/dtc-guide.html`)

- [ ] External CSS `guides-dtc.css` loads (TOC sidebar, concept maps styled)
- [ ] Theme toggle; dark mode readable on diagrams
- [ ] TOC open/close on mobile

## D. SF 28 + Corporate structures

- [ ] `guides-sf28.css` / `guides-corporate.css` linked and applied
- [ ] No missing styles after extraction (header, TOC, quiz, modals)

## E. Deploy / production parity

- [ ] Local commit is on `main` and pushed (or PR merged)
- [ ] Production URL shows new commit (check OA Canon section + CSS file network 200)
- [ ] Hard refresh production after deploy to beat HTML cache

## F. Search index (after content edits)

```bash
python3 scripts/build-search-index.py
npm test   # optional
```

- [ ] Hub search finds new modal/section titles
- [ ] `js/search-index.js` header says regenerated by the script

---

## Fail criteria (block deploy)

1. CUSIP values shifted one column / overflow outside form  
2. OA Canon JSON 404 on production  
3. Unstyled page (CSS 404 after path change)  
4. Scenario tabs do not switch  
5. Broken theme (white text on white / black on black on primary content)
