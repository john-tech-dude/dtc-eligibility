## How to Use

Open `index.html` in any modern web browser to access the complete documentation collection. From the main index, you can navigate to any document. No server or build process required.

### Navigation

- **Main Index**: Start at `index.html` for document collection overview
- **Desktop**: Click the sidebar tab or press `Ctrl/Cmd + B` to toggle table of contents
- **Mobile**: Tap the TOC button to open the navigation sidebar
- **Links**: All section links open in the same tab for seamless reading
- **Cross-references**: Each document includes links to related documents
