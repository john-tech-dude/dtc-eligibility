# NIIDS teaching add-on (OA § I.C.3)

**Source:** DTC Operational Arrangements as of June 10, 2026, § I.C.3 + Exhibit B Note 2  
**Teaching surface:** `pages/forms/dtc-eligibility-questionnaire.html` v1.4  

## Teaching points shipped

1. **Definition:** New Issue Information Dissemination Service in **UW SOURCE** for munis  
2. **Who:** Dissemination Agents (underwriters / authorized reps)  
3. **How:** Input NIIDS Data Elements → **Disseminate Security Information** → data to NIIDS Subscribers  
4. **If not disseminated:** No dissemination if data not input or function not selected  
5. **Purpose:** Trade reporting, matching, confirmation (MSRB-related); element list on DTCC.com  
6. **Separate from eligibility:** Overlap possible; completion of one ≠ the other (OA explicit)  
7. **DTC role:** Conduit only; as-is / no validation (condensed legal notice)  
8. **Exhibit B:** Muni trade/settlement-eligible data via NIIDS
